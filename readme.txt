Before to run the code you want to do:
1) Create VK Web Application:
	a) Log in to VK with your user
	b) Go to http://vk.com/dev -> Create an Application
	c) Enter the "Title" of your application and then choose in the "category" options "Website"
	d) Enter your web site address to the "Site address" field
	e) Enter your web site base domain name to the "Base domain" field
2) After you create your VK Web App 
	a) Go to the "Settings" and copy your "Application ID" number. 
3) Go to js/VKapp.js
	a) Line: 16, replace API_ID to your Application ID number which you coped in step 2.a


That is it. Now you can use it
